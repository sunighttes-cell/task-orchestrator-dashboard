# Task Orchestrator Dashboard

Full-stack orchestration dashboard built with:

## Frontend
- React
- TypeScript
- Vite
- React Query
- Zustand
- TailwindCSS

## Backend
- Java 21
- Spring Boot 3
- Spring Data JPA
- PostgreSQL

## Features
- Job tracking
- Retry workflows
- Status dashboards
- Polling updates
- Search/filtering

## Running Locally

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
./mvnw spring-boot:run
```
